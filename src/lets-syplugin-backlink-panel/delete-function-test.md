# SQL删除功能测试

## 测试目的
验证新增的SQL删除功能是否正常工作。

## 测试步骤

### 1. 基础删除测试
1. 在SQL面板中输入一个SQL语句
2. 点击"保存SQL"按钮，输入名称"测试删除1"
3. 再次点击"删除SQL"按钮
4. 输入名称"测试删除1"
5. 点击"确认删除"
6. 验证删除成功提示和SQL确实被删除

### 2. 不存在SQL删除测试
1. 点击"删除SQL"按钮
2. 输入一个不存在的SQL名称"不存在的SQL"
3. 点击"确认删除"
4. 验证显示"未找到名为...的SQL配置"提示

### 3. 取消删除测试
1. 点击"删除SQL"按钮
2. 输入SQL名称
3. 点击"取消"按钮
4. 验证表单关闭且没有删除操作

### 4. 多SQL删除测试
1. 保存多个SQL（如"测试1"、"测试2"、"测试3"）
2. 删除其中一个
3. 验证其他SQL保持不变

### 5. 空名称删除测试
1. 点击"删除SQL"按钮
2. 不输入任何内容直接点击"确认删除"
3. 验证显示"请输入要删除的SQL名称"提示

## 预期结果
- 删除功能正常工作
- 有适当的确认和错误提示
- UI响应流畅
- 删除操作有明确的视觉反馈

## 测试环境
- 浏览器：Chrome/Safari
- 操作系统：macOS
- 测试时间：2025-12-20

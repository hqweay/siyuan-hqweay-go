## 0.24.0

### Minor Changes

- - 反链面板：设置页面 UI 现代化重构，采用更精致的圆角、阴影和过渡动画，并保持与思源自带主题色的完美兼容。
  - 反链面板：加入极致的“按需懒加载”优化。配置文档底部默认折叠时，将阻断所有的冗余 SQL 查询和组件渲染，实现零加载负担；展开后再次折叠时也能完美保留搜索与过滤状态。默认行为已调整为折叠。

## 0.23.0

### Minor Changes

- - 反链面板：进行深度 SQL 性能优化，消除全表扫描和子查询冗余，显著降低加载超大文档时的磁盘 I/O 和卡顿。
  - 全局查询：引入智能缓存机制，在切换文档或操作查询面板时响应更丝滑；修改块内容后会自动刷新，兼顾性能与实时同步。

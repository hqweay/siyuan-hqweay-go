# 行内元素展示插件

这个插件会在思源笔记的 dock 栏中展示当前打开文档的所有行内标注元素（使用 `==文本==` 格式的内容）。

## 功能特性

- **实时展示**: 自动获取当前文档中的所有行内标注
- **搜索过滤**: 支持对标注内容进行实时搜索
- **上下文显示**: 显示标注所在段落的上下文
- **快速操作**: 支持复制标注内容和跳转到对应位置
- **自动更新**: 文档切换时自动更新显示内容

## 使用方法

1. 在思源笔记中启用此插件
2. 在右侧 dock 栏中找到"文档标注"面板
3. 查看当前文档中的所有标注内容
4. 使用搜索框快速查找特定标注
5. 点击复制按钮复制标注内容
6. 点击跳转按钮定位到标注在文档中的位置

## 技术实现

- 使用 SQL 查询从数据库中提取包含 `==` 标记的块
- 通过正则表达式提取具体的标注内容
- 监听文档切换事件实现响应式更新
- 使用 Svelte 构建用户界面

## 快捷键

- `⌥⌘W`: 打开/关闭文档标注面板

## 注意事项

- 插件仅显示使用 `==文本==` 格式的标注
- 每次文档切换时自动刷新内容
- 支持多个标注的并行显示

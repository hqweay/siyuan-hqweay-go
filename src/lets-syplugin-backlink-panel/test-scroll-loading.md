# 滚动加载功能测试

## 功能描述
本测试用于验证反链面板的滚动加载功能是否正常工作。

## 测试步骤

### 1. 启用滚动加载
1. 打开设置面板
2. 找到"滚动加载"选项
3. 确保该选项已启用

### 2. 创建测试数据
1. 创建或打开一个文档
2. 添加多个反链引用（至少20-30个）
3. 确保有足够的内容触发滚动

### 3. 测试滚动加载
1. 打开反链面板
2. 观察初始加载的项目数量
3. 滚动到面板底部
4. 验证是否自动加载更多数据
5. 检查加载状态指示器是否显示

### 4. 测试状态管理
1. 在滚动加载过程中更改筛选条件
2. 验证状态是否正确重置
3. 确认新搜索从第一页开始

## 预期结果
- 初始显示少量项目
- 滚动到底部时自动加载更多
- 加载状态有明确指示
- 状态变更时正确重置

## 已知问题
如果滚动加载不工作，请检查：
1. 设置中是否启用了滚动加载
2. 浏览器控制台是否有错误
3. 网络请求是否正常发送

## 故障排除
如果遇到问题，请：
1. 清除浏览器缓存
2. 重新加载插件
3. 检查设置配置

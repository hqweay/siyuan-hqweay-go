# 恐龙工具箱 (Dinosaur Toolbox)

> 这是一只恐龙，路过的人必须夸它可爱。🦖
>
> 恐龙工具箱是一款思源笔记增强插件，采用模块化**子插件（Sub-Plugin）自动注册机制**，将各项独立的实用功能高度集成在一起。

---

## 🛠️ 核心功能一览

目前恐龙工具箱内置了 **22 个** 相互独立的子插件功能，覆盖了日常编辑、多端同步、排版优化、阅读批注和视图转换等各个场景。

### 🌟 重点核心功能

#### 1. 📂 反链面板增强 ([lets-syplugin-backlink-panel](./src/lets-syplugin-backlink-panel))
*基于 `Misuzu2027/syplugin-backlink-panel` 增强*
- **自定义 SQL 展示**：支持在反链面板中自定义并展示 SQL 查询结果。
- **Tab 页签持久化**：记住在文档中最后打开的页签（Tab），下次进入该文档时自动打开。
- **文档流优化**：提升了文档流浏览的使用体验。

#### 2. 📊 仪表盘与文档流 ([lets-dashboard](./src/lets-dashboard))
- **多功能仪表盘**：专为日记与回顾设计的仪表盘视图，兼容移动端与 PC 端。
- **文档流 & 图片流**：支持纯文本或纯图片的流式浏览视图。
- **Schema 链接唤起**：支持通过 `siyuan://` 链接直接在思源中打开指定的 Flow 界面、图片流或特定仪表盘。

#### 3. 🧭 导航栏助手 ([lets-nav-helper](./src/lets-nav-helper))
- **双端形态**：PC 端支持可展开的玻璃质感悬浮按钮 (FAB)，移动端提供现代化的悬浮胶囊导航条，全面适配思源极简主题。
- **原生图标**：全面接入思源内置的 `litheness` SVG 图标库，视觉体验高度原生化。
- **快捷操作**：新增“快捷操作”配置组件，支持自定义菜单以快速打开网页、随机漫游、新建日记或执行复杂 SQL 查询。
- **上下文切换**：快捷在相邻的两个文档之间进行切换。

#### 4. 📖 EPUB 书籍阅读 ([lets-epub-reader](./src/lets-epub-reader))
- **批注双链化**：阅读 EPUB 电子书时，选中文本进行标注，会自动将标注插入至绑定的思源文档中作为引用。
- **悬浮窗笔记**：自动弹出当前标注的思源悬浮窗，支持移动端随时记录阅读笔记。

#### 5. 🔍 OCR 识别助手 ([lets-ocr](./src/lets-ocr))
- **自动 PDF OCR**：在阅读 PDF 并进行矩形标注时，复制图片后粘贴自动进行 OCR 并粘贴为文本（需在“粘贴时数据处理”中开启）。
- **一键图片 OCR**：支持右上角栏一键对当前文档内的所有图片进行 OCR 识别。
- **快捷菜单**：为图片的右键菜单增加“快捷复制 OCR 文本”和“使用 OCR 文本替换图片”选项。

#### 6. 📅 日记辅助工具 ([lets-how-to-write-diary](./src/lets-how-to-write-diary))
- **全局小窗日记**：支持全局快捷键（默认 `F10`）弹出一个便捷小窗口，快速记录今日日记（参考 `suka233/siyuan-knote`）。
- **时间旅行式双链**：Slash 命令 `/cdn` 或 `创建日记引用` 呼出日历选择器，可快速创建任意指定日期（包括未来或过去日期）的日记文档并插入为块引用，实现“向未来日期传递信息”的双链闭环（参考 `frostime/siyuan-plugin-kits`）。
- **天气自动写入**：自动获取当前天气并写入文档自定义属性，可在日记标题下方实时展示。

#### 7. 🏷️ 属性与视图转换 ([lets-block-attr](./src/lets-block-attr))
- **快捷添加属性**：块菜单中支持一键赋予多个属性。以 `@` 开头命名的配置可直接注册为 Slash 命令调用。
- **视图一键转换**：支持添加自定义属性将思源列表一键转换为表格、思维导图、时间线、看板、Tab 等多种视图效果（参考 `Achuan-2/siyuan-themes-tsundoku`）。
- **便利贴样式**：启用代码片段中的便利贴，使用自定义属性 `custom-f`（如 `qsy` 浅黄, `qsg` 浅绿等）为块注入便利贴样式 CSS（参考 `chenshinshi/sy-blt`）。

#### 8. 🔊 VoiceNotes 同步 ([lets-voicenotes-sync](./src/lets-voicenotes-sync))
- **云端笔记同步**：自动/手动同步 VoiceNotes (voicenotes.com) 的语音笔记至指定笔记本和目录下。
- **格式化与过滤**：支持同步时自动中英文排版优化、换行新建块处理，并可配置 comma 分隔的排除标签过滤笔记。

### 🛠️ 实用辅助与体验增强功能

#### 9. ✏️ 标题层级转换 ([lets-adjust-title](./src/lets-adjust-title))
- 在块上下文菜单或文档标题菜单中快捷转换所有标题或选定标题的层级。

#### 10. 📋 粘贴数据预处理 ([lets-do-on-paste](./src/lets-do-on-paste))
- 粘贴内容时自动进行格式化预处理（如自动获取网页标题链接、图片粘贴自动缩小为 50%、清除多余空行等）。

#### 11. 🚪 边栏自动开关 ([lets-dock-show-and-hide](./src/lets-dock-show-and-hide))
- 打开特定文档时，自动控制左/右侧边栏（Dock）及停靠栏的显示和隐藏（支持配置默认宽度、分屏自动忽略）。

#### 12. ➕ 边栏功能扩充 ([lets-dockPlus](./src/lets-dockPlus))
- 可在左/右边栏的顶部或底部添加自定义图标，用于快速打开指定链接、执行特定 SQL 或固定文档。

#### 13. 📌 侧边栏内容展示 ([lets-docky](./src/lets-docky))
- 在侧边栏的指定位置展示指定的文档或块内容，支持指定自定义图标、快捷键及缩放比例。

#### 14. 📑 代码片段托管 ([lets-fetch-code-snippets](./src/lets-fetch-code-snippets))
- 托管与管理收集的 CSS/JS 代码片段，支持从 GitHub Issues 自动拉取并在本地快速启用。

#### 15. 🔗 行内元素转换 ([lets-href-to-ref](./src/lets-href-to-ref))
- 块菜单/文档菜单中支持引用⇄块超链接互转、清理自身引用、清理 * 引用，以及一键将粗体/标注/标签/斜体/下划线/删除线/上标/下标共 **10 种行内元素** 转为纯文本。
- **全文命令**：13 个操作均已注册到命令面板（⌘/）和思源快捷键设置（共 3 组互转 + 10 组转文本），采用 SQL 批量查块 ID + `getBlockDOM` API 实现真正的全文处理，无前端懒加载遗漏。

#### 16. 📤 提取行内元素 ([lets-inline](./src/lets-inline))
- 一键提取文档中的所有标注、备注等行内元素至一个新文档中（支持以大纲块格式或携带上下文环境导出）。

#### 17. 📝 侧边栏展示高亮和备注 ([lets-inline-elements](./src/lets-inline-elements))
- **多模式展示**：在侧边栏面板中直接列出当前文档的所有 `==highlight==` 高亮标注与行内备注 `<sup>(备注)</sup>`。
- **直接可编辑**：支持简单模式（快速浏览与搜索）和 Protyle 模式（支持直接编辑标注和备注内容）双视图一键切换。
- **定位与跳转**：支持搜索过滤、复制 ID、滚动分页加载以及双击直接跳转至文档内对应位置。

#### 18. 🖼️ 随机题头图 ([lets-random-header-image](./src/lets-random-header-image))
- 随机为文档设置题头图，支持本地 Siyuan 文件夹、必应每日壁纸、岁月小筑及其他自定义图片 API。

#### 19. 👓 阅读体验增强 ([lets-read-helper](./src/lets-read-helper))
- 在思源自带的阅读工具栏中新增“标注并复制块引”快捷动作（支持 Text*、* 等格式）。

#### 20. 📤 发送到与卡片分享 ([lets-send-to](./src/lets-send-to))
- 将内容发送到剪贴板或第三方应用。内置精美的卡片分享功能，支持自定字体、段落间距、底色和自定义卡片模板。

#### 21. 📐 中文排版优化 ([lets-typography](./src/lets-typography))
- 文档或选定块的一键格式化排版，支持自动在中英文间加空格、网络资源图片本地化及图片居中等。

#### 22. 🔖 快捷书签添加 ([lets-bookmark](./src/lets-bookmark))
- 块上下文菜单中新增快捷添加到指定书签的选项（可在设置中按行配置常用书签分类）。

## 🏗️ 架构与二次开发

该项目实现了一套优雅的**子插件自动注册机制**（PluginLetsGo），将各个独立的功能模块隔离开来：
- 只要在 `src/lets-*` 目录下创建功能模块并实现 `index.ts`（继承 `SubPluginBase`）和 `plugin.ts`，插件便会自动扫描并注册该功能。
- 所有翻译采用模块化设计，集中在 `src/translations/parts/` 下，构建时会自动合并。
- 更多开发约束与设计规则详见 [AGENTS.md](./AGENTS.md)。
